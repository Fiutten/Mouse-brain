\section{Research contributions}
\label{sec:contributions}

The main contribution is not a new mouse-brain simulator. It is a falsifiable
procedure for determining what kind of claim is supported by a partial digital
model. This contribution is developed through the following elements:

\begin{itemize}
  \item A claim-audit formulation that separates prediction, reproducibility,
  topology specificity, directed identifiability, local structure-function
  association, and computational cost. The output is not only a ranking: it is
  an explicit set of admitted and rejected interpretations.

  \item A formal \acs{mis} composed of non-interchangeable evidence blocks and fixed
  directional thresholds. The rule is tested with four known-truth systems that
  isolate common drive, topology without direction, and direction without
  topology specificity.

  \item A cross-resource protocol that assigns Allen, Sensorium, Dynamic
  Sensorium, and \acs{microns} distinct roles instead of treating heterogeneous data
  as interchangeable validation. This design provides a real negative control,
  predictive controls, and a local anatomical-functional test.

  \item An external-validity case in \acs{microns} obtained with a specified primary
  endpoint, distance- and degree-matched null models, false-discovery-rate
  correction, unit-cluster bootstrap intervals, and two non-overlapping hold-out
  cohorts. It tests reproducible recovery of a published broad phenomenon rather
  than claiming a new biological wiring rule.

  \item A reproducible publication freeze that binds numerical artifacts to
  allowed and blocked claims. In the current study it admits a replicated local
  observational structure-function association, while rejecting causal,
  behavioural, whole-brain, and state-of-the-art predictor claims.
\end{itemize}

These elements define the scope of the work. MouseBrainBench evaluates the
relationship between evidence and interpretation. It does not establish that a
single score is universally sufficient for mechanistic neuroscience. The \acs{mis}
thresholds and evidence blocks must be specified for the target claim before
evaluation, and failure means that the particular claim is unsupported, not
that the dataset or model is devoid of scientific value.
