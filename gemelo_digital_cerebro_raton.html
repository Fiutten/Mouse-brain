Plan critico: gemelo digital modular de cerebro de raton

Propuesta multiescala para modelar dinamicas regionales, integracion funcional, memoria activa y seleccion de accion sin afirmar conciencia subjetiva.

Grupo: DSLab

Fecha: 3 de junio de 2026

Especie propuesta: Mus musculus, raton de laboratorio adulto

Version: 0.1 - marco critico inicial

Decision Principal

La especie base debe ser Mus musculus, raton de laboratorio adulto.

Motivo: es el mamifero con mejor ecosistema abierto para construir un modelo defendible: Allen Mouse Brain Common Coordinate Framework, Allen Mouse Brain Connectivity Atlas, Allen Brain Observatory, Visual Behavior Neuropixels, bases de datos de tipos celulares, transcriptomica, morfologia, electrofisiologia y literatura abundante sobre corteza, talamo, hipocampo, ganglios basales, cerebelo y tronco.

No elegimos reptil como especie principal. Aunque tiene interes evolutivo y una organizacion pallial distinta, hay menos datos estandarizados, menos conectividad sistematica y menos benchmarks funcionales. Usarlo ahora aumentaria el riesgo de construir una maqueta especulativa.

Pregunta Cientifica Defendible

No vamos a intentar demostrar "conciencia digital".

Que arquitectura neurocomputacional minima, organizada por regiones cerebrales de mamifero, reproduce integracion sensoriomotora, memoria activa, seleccion flexible de accion y estados globales comparables a correlatos funcionales de acceso consciente?

Esto permite hablar de integracion global, acceso/broadcasting, estados de vigilia/sueno simplificados, atencion, memoria de trabajo, seleccion de accion, perturbacion y recuperacion del sistema. No permite afirmar experiencia subjetiva.

Tesis Provisional

Un modelo modular por regiones, calibrado con conectividad mesoscale y dinamica neural simplificada, puede reproducir propiedades funcionales asociadas a acceso global y control adaptativo con menos detalle celular del que requiere una emulacion completa.

La contribucion potencial no seria "hemos creado conciencia", sino una arquitectura regional biologicamente restringida, un protocolo de validacion contra datos reales y tareas conductuales, un analisis de que modulos son necesarios para integracion global y conducta flexible, y una ruta incremental hacia mayor detalle biologico.

Nivel de Escala

Nivel 0: Grafo regional funcional

Cada region cerebral es un nodo dinamico. Las conexiones son pesos dirigidos derivados de conectividad mesoscale. La dinamica puede ser neural mass, rate model o sistema de estado, con entrada sensorial y salida motora simplificadas.

Regiones iniciales: corteza visual primaria y areas visuales superiores, talamo visual, hipocampo, corteza prefrontal/association cortex aproximada, ganglios basales, amigdala, hipotalamo, cerebelo y tronco encefalico/arousal system.

Objetivo: tener un sistema cerrado que percibe, mantiene estado interno, elige accion y cambia de regimen dinamico.

Nivel 1: Poblaciones neuronales por region

Cada region contiene poblaciones excitatorias, inhibitorias y moduladoras. La dinamica puede ser spiking LIF/AdEx o rate-spiking hibrida, con conectividad interregional fija o parcialmente aprendible y neuromodulacion global simplificada.

Nivel 2: Microcircuitos seleccionados

No detallaremos todo el cerebro. Elegiremos regiones criticas: V1/talamo visual para entrada sensorial, hipocampo para memoria espacial/contextual, ganglios basales para seleccion de accion y, posteriormente, corteza prefrontal/association cortex para memoria de trabajo.

Nivel 3: Gemelo digital calibrable

Solo cuando los niveles anteriores funcionen. El objetivo sera inferir parametros desde datos reales, ajustar instancias de modelo y validar prediccion de actividad retenida, generalizacion a estimulos no vistos y efectos de perturbaciones.

Arquitectura Funcional Inicial

  • Sensorial: entrada visual simplificada y codificacion temporal/event-based opcional.
  • Talamo: relay, gating sensorial y control de ganancia.
  • Corteza sensorial: extraccion de caracteristicas y procesamiento recurrente local.
  • Hipocampo: memoria de estado/contexto y asociacion episodio-lugar-objetivo.
  • Ganglios basales: seleccion de accion, aprendizaje por recompensa y competencia entre politicas.
  • Arousal/neuromodulacion: alerta global, dopamina para error de recompensa y variables simplificadas de incertidumbre y ganancia.
  • Workspace funcional: mecanismo de acceso global entre percepcion, memoria y accion, sin presentarlo como conciencia.

Tareas Iniciales

  • Decision visual con memoria corta: estimulo visual, retardo, eleccion y recompensa.
  • Navegacion 2D simplificada: agente en entorno pequeno, claves visuales, objetivo cambiante y memoria contextual.
  • Reversal learning: cambio de regla de recompensa para medir adaptacion, perseveracion y recuperacion.
  • Lesion virtual: apagar hipocampo, talamo, ganglios basales o workspace y comprobar deficits esperados.

Criterios de Exito

  • Reproducir patrones cualitativamente esperados tras lesion.
  • Mejorar sobre ablations no regionales.
  • Mantener conducta flexible bajo cambio de regla.
  • Generar dinamicas internas interpretables.
  • Permitir mapear variables del modelo a regiones/circuitos.
  • No depender de ajuste manual fragil.

Criterios de Fracaso

  • El modelo solo funciona por aprendizaje end-to-end sin que la arquitectura regional importe.
  • Las regiones son decorativas y no explican conducta.
  • No hay datos suficientes para calibrar o validar.
  • La dinamica interna no es mas informativa que una RNN/Transformer pequena.
  • El argumento de conciencia empieza a depender de retorica en vez de mediciones.

Stack Tecnico Recomendado

Fase inicial: Python, PyTorch, NumPy/SciPy, NetworkX, Gymnasium o entorno 2D propio, Matplotlib/Plotly y AllenSDK para datos cuando empiece la calibracion.

Fase spiking: snnTorch o SpikingJelly para SNN entrenables, Brian2 para simulacion neurocientifica explicita y Brian2CUDA solo si necesitamos escala GPU.

Declaracion Sobre Conciencia

No afirmaremos que el sistema es consciente. Podremos estudiar acceso global, integracion funcional, memoria de trabajo, sensibilidad a perturbacion, estados globales, reportabilidad computacional y correlatos dinamicos inspirados en teorias de conciencia.

El modelo permite investigar condiciones funcionales y arquitectonicas asociadas a correlatos de acceso consciente en cerebros de mamifero, sin asumir ni demostrar experiencia subjetiva.

Primer Entregable Recomendado

Construir un prototipo Nivel 0: grafo regional de raton simplificado, dinamica regional, tarea visual con memoria y seleccion de accion, lesiones virtuales, visualizacion de actividad por region y comparacion contra un baseline no modular.

Si Nivel 0 no muestra ventajas interpretables frente a un baseline simple, no debemos pasar a niveles mas biologicos.
