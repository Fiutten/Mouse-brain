\section{Data and code availability}

The MouseBrainBench source code and generated benchmark artifacts are maintained
in the project repository. Raw public datasets are not redistributed. Data
acquisition scripts and provenance records identify the public sources used
where licenses and application programming interfaces allow reproducible access.
MICRONS CAVE access may require a CAVE token and acceptance of the corresponding
terms of service. A release DOI, final repository URL, and exact submission
commit will be added when the manuscript is frozen for submission.
